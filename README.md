# Sudoku

## Quickstart
```shell
$ ./build.sh && ./main
```

## Keys

| Key | Description |
|-----|-------------|
| <kbd>h</kbd> | Move left |
| <kbd>j</kbd> | Move down |
| <kbd>k</kbd> | Move up |
| <kbd>l</kbd> | Move righ |
| <kbd> </kbd> | Clear cell |
| 1-9 | Fill cell |
| <kbd>q</kbd> | Quit |
